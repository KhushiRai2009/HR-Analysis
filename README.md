# HR-Analysis
HR Analytics Dashboard built using Tableau, focusing on employee performance, attrition trends, salary distribution, recruitment funnel, and workforce demographics. Includes KPIs, interactive filters, and actionable insights for HR decision-making.
HR Analytics Dashboard (Tableau)

This repository contains a concise and interactive HR Analytics Dashboard developed in Tableau. The project summarizes key workforce metrics to help organizations monitor employee trends and make data-driven HR decisions.

Overview

The dashboard analyzes attrition, employee demographics, salary patterns, and recruitment performance. It highlights critical HR insights through clear KPIs and interactive visualizations.

Key Features

Attrition analysis by department, job role, age group, and salary band

Workforce demographics including gender, education, and age segmentation

Salary and performance insights across roles and levels

Recruitment funnel view and hiring trend analysis

KPI summary for total employees, attrition rate, average salary, and tenure


Dataset
Includes fields such as age, gender, education, job role, department, monthly income, tenure, performance rating, and attrition status.

Tools Used

Tableau for visualization

Used Data from kaggle
